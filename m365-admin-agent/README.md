# M365 Admin Agent

Enterprise AI-powered Microsoft 365 administration assistant.

## Features

- React + TypeScript frontend
- Azure Static Web Apps
- Azure Functions backend
- Entra ID authentication
- Azure AI Foundry integration
- Exchange / SharePoint / Teams / Viva administration

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Run Backend

```bash
cd backend
pip install -r requirements.txt
func start
```
